Plugin WooCommerce para la pasarela de pagos Tuyo Pay.

== Description ==

Plugin para integrar la pasarela de pagos Tuyo Pay en el sistema de carrito de compras de WooCommerce de WordPress.

REQUISITOS
-PHP version 7.1+
-WordPress 5.4+
-WooCommerce 3.5+

== Installation ==

1. Descargue el archivo ZIP con la versión más reciente del plugin.
2. Ingrese al panel administrativo de su WordPress.
3. Ingrese en la opción del menú Plugins > Añadir Nuevo > Subir Plugin.
4. Busque el plugin con formato .zip descargado en su computadora y subirlo como cualquier otro archivo.
5. Luego de instalar el plugin se podrá ver en el listado de plugins instalados.
6. Para configurar el plugin se deberá de ingresar en la opción del menú: Woocommerce > Ajustes > Pagos > Tuyo Pay
7. Ingrese los API Keys proporcionados por su KAM asignado en los respectivos campos de texto.
8. Configurar los demás campos como usted desee o como lo haya descrito el KAM.
9. ¡Listo!
